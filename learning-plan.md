## Fixed Entry Points

- Handbook: https://aiweb3.school/zh/handbook/
- WCB Course: https://web3career.build/zh/programs/AI-Web3-School
- WCB Learning: https://web3career.build/zh/programs/AI-Web3-School#tab=learning

## Daily Routine

1. Check WCB Learning for today's course, task, meeting, and check-in entry.
2. Read the relevant Handbook chapter.
3. Pick one path:
   - Minimum path: finish today's essential reading/task.
   - Recommended path: add notes, examples, and questions.
   - Challenge path: build a small experiment or publishable artifact.
4. Write `daily/YYYY-MM-DD.md`.
5. Generate check-in draft and submit manually on WCB/check-in platform.
6. Record submission link or proof in the daily note.

## Weekly Review

- Summarize completed tasks.
- Collect open questions.
- Move Handbook issues to `handbook-feedback/`.
- Identify one reusable artifact to polish.
