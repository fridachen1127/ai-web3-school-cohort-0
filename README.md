Personal learning journal and proof-of-work for **AI × Web3 School**.

- WCB Agent API Docs: https://web3career.build/llms.txt

## Privacy Reminder

This repository is public by default. Do **not** commit or publish:

- API keys, tokens, passwords, or secrets
- Wallet seed phrases, private keys, or keystore files
- Unpublished personal contact information
- Internal meeting links or private course links
- Other people's personal data

## Directory Structure

- `profile.md` — learner profile and preferences
- `learning-plan.md` — personal learning plan and weekly focus
- `daily/` — daily learning notes and check-in drafts
- `tasks/` — course tasks and proof-of-work notes
- `experiments/` — small AI/Web3 experiments and prototypes
- `handbook-feedback/` — feedback, questions, typos, unclear concepts, outdated resources
- `hackathon/` — hackathon ideas, plans, demos, and retrospectives
- `submissions/` — submitted check-ins, task links, and proof records
- `templates/` — reusable daily/task note templates

## Working Principle

This repo is a public proof-of-work workspace: keep notes clear, reusable, linkable, and safe to share.
